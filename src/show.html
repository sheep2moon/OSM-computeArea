Usługa sieciowa typu REST API wykorzystująca zapytanie HTTP GET. Dane udostęniane są w formacie JSON, XML, CSV, JSV.

\r\n\r\n

Przykładowe zapytania:

\r\n\r\n

1. Pobieranie obiektu - otwarteDaneZestawienieZrn/{klucz}?{format}

\r\n\r\n

Jako klucz ustawione jest pole inspireId, np.

\r\n\r\n

https://api.zabytek.gov.pl/nidrestapi/api/data/geoportal/otwarteDaneZestawienieZrn/PL.1.9.ZIPOZ.NID_N_06_ZE.1365?format=json

\r\n\r\n

2. Filtrowanie obiektów -  filter=[\"Amount\", \">\", 20] np.

\r\n\r\n

https://api.zabytek.gov.pl/nidrestapi/api/data/geoportal/otwarteDaneZestawienieZrn?format=json&filter=[\"wojewodztwo\",\"=\",\"lubelskie\"]

\r\n\r\n

3. Pobieranie określonej liczby obiektów - take=value np.

\r\n\r\n

https://api.zabytek.gov.pl/nidrestapi/api/data/geoportal/otwarteDaneZestawienieZrn?format=json&take=10

\r\n\r\n

4. Pomijanie rekordów – skip=value np.

\r\n\r\n

https://api.zabytek.gov.pl/nidrestapi/api/data/geoportal/otwarteDaneZestawienieZrn?format=json&take=100&skip=5000

\r\n\r\n

5. Sortowanie – sort={\"selector\":\"nazwa_atrybutu\", desc:false} np. 

\r\n\r\n

https://api.zabytek.gov.pl/nidrestapi/api/data/geoportal/otwarteDaneZestawienieZrn?format=json&take=100&sort={\"selector\":\"wojewodztwo\", desc:true}
